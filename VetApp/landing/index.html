Bienvenido a VetApp

Vet App, es una aplicación web que cuenta con formularios especialmente diseñados para la carga de datos de las mascotas (el seguimiento de sus tratamientos, calendario de vacunación, estudios y prácticas) y de sus dueños.

VetApp como herramienta

Esta aplicación es de suma utilidad para el veterinario en la práctica cotidiana, al tener centralizada toda la información necesaria para su asistencia. Fácil de actualizar y evita el extravío de estudios que podrían ser relevantes para el diagnóstico.
